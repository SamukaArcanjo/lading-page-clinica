# 🦷 Landing Page - Clínica Odontológica

Este projeto é uma landing page moderna e responsiva desenvolvida para uma clínica odontológica, com foco em conversão de pacientes e apresentação profissional dos serviços.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- jQuery

## 🎯 Funcionalidades
- Layout responsivo (adaptado para desktop e mobile)
- Menu interativo com indicador animado
- Header fixo durante a navegação
- Seções organizadas: Home, Sobre e Tratamentos
- Botões de chamada para ação (CTA) para agendamento
- Menu mobile com interação dinâmica

## 💡 Objetivo do projeto
Criar uma interface moderna e intuitiva para clínicas odontológicas, com foco em experiência do usuário (UX) e aumento de conversão.
